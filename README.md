## **Gemini AI integration for block game.**

## Dependences:
- Java 21+
- Paper/Spigot/anything can run plugin with 1.21+ (older version not tested)

## Usage:
- /gemini (Display plugin info)
- /gemini ask <prompt> (Send prompt to gemini)
- /gemini toggle (toggling to chat directly into Gemini)
- /gemini enable (Enable this plugin)
- /gemini disable (Disable this plugin)
- /gemini help (Show usage on this plugin)
